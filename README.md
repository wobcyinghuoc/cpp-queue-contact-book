# 循环队列简易通讯录
数据结构课程设计，使用纯C++手动实现循环队列，完成联系人基础管理功能。

## 项目功能
- 新增联系人
- 删除指定联系人
- 修改联系人电话
- 按姓名查找联系人
- 展示所有联系人

## 技术实现
- 底层结构：手写循环队列，数组模拟实现
- 核心逻辑：队列初始化、判空、判满、取模循环遍历
- 本程序最多支持存储 29 个联系人（循环队列牺牲 1 个位置区分空 / 满状态），支持中文姓名和 11 位手机号输入
- 纯原生代码编写

## 运行环境
- 编程语言：C++
- 编译工具：GCC
- 使用方式：编译后在控制台直接运行
